### Giving Back Through Code: Motion-U Visits CFS to Teach Git Basics

**14 April**

On **14 April**, members of **Motion‑U** took a meaningful step in giving back to the community by visiting **Centre for Foundation Studies (CFS)**. The goal was simple but impactful — to guide foundation students through the basics of **Git**, an essential tool every modern developer should know.

For many foundation students, Git was a completely new concept. Motion-U members started from the very beginning, explaining what version control is, why it matters, and how Git is used in real-world software development. Instead of overwhelming students with theory, the session focused on practical understanding and hands-on demonstrations.

The workshop covered core fundamentals such as:

* What Git is and how it helps developers work in teams
* Basic commands like `git init`, `git add`, `git commit`, and `git status`
* Understanding repositories and tracking code changes

What made the session special was the relaxed and friendly environment. Motion-U members shared their own early experiences learning Git — including common mistakes — which helped foundation students feel more confident and less intimidated by the tool.

Beyond just teaching commands, the session aimed to spark curiosity and confidence. By introducing Git early, Motion-U hopes foundation students will be better prepared when they enter their degree programs, especially those pursuing computer science or related fields.

This visit reflects Motion-U’s core mission: building a strong developer culture by sharing knowledge, supporting juniors, and growing together as a tech community. It wasn’t just about teaching Git — it was about inspiring the next generation of developers to start their journey with the right mindset and tools.

Small steps like this create lasting impact, and Motion-U looks forward to continuing similar initiatives in the future.
